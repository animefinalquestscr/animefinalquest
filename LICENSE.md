## 🛠️ Troubleshooting Common Issues

- ❌ **Script GUI does not appear after execution**
  The overlay may be hidden behind the game window. Try pressing the GUI toggle key (default `RightShift`). If the issue persists, confirm your executor is fully injected before pasting the **Anime Final Quest Script**.
  **Solution:** Re-inject the executor after the game has fully loaded past the loading screen, then execute the script again.

- ❌ **Kill Aura stops targeting mobs randomly**
  This usually happens when the mob IDs change after a game update, causing the aura to fail its target checks.
  **Solution:** Update to the latest version of the **Anime Final Quest Script Kill Aura** module, which automatically re-maps mob IDs on each session start.

- ❌ **Auto Farm loop gets stuck on one quest**
  The quest objective anchor may have shifted coordinates in a recent map update.
  **Solution:** Switch `autoQuestMode` to `"sequential"` in your `config.json` and restart the script to force a fresh quest chain.

- ❌ **Lag spikes when using Mob Magnet**
  Pulling too many entities at once can overload the render pipeline on lower-end hardware.
  **Solution:** Reduce `mobMagnetRadius` to `20` or lower in your config and disable **Stat Viewer** to free up frame budget.

- ❌ **Script crashes on mobile after a few minutes**
  Mobile executors have stricter memory limits that the default config can exceed.
  **Solution:** Set `loopDelay` to `500` and disable Inventory Manager to reduce background processing load on Android and iOS.
